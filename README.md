Copcat IMAP
=============

This is a simple script to move messages from one email inbox to another.
-------
    copycat.py:
    Please provide a few tidbits of data to run:
        --from_user id1  
        --from_pw   pw1  
        --from_host host1
        --to_user   id2  
        --to_pw     pw2  
        --to_host   host2
